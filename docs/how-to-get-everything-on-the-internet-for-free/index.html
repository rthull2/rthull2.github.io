How to Get Everything on the Internet for Free.

Robert Hull

28 Nov 2021 • 2 min read

Free books, TV shows, movies, courses, applications, operating systems, etc. Everything is free if you know what you're doing. I'm going to give you a brief tutorial.

Actually Free

There are plenty of business models that give you stuff for free. The most common type is an ad-based model, which becomes completely free after you start using an ad blocker. Donation based non-profits are free if you don't give them any money.

The only trick with this category is that you need to know about the things that are free. Some people are still paying for Microsoft Word which Google Docs or LibreOffice would work fine. Finding free things is often as simple as doing a web search for "how to get X for free?".

Free Trials (or Free Returns)

Services such as Netflix and Amazon Prime will often offer multiple one-month trials for the same account. If you make multiple accounts (not sure if they check to see if you're using the same credit card, but getting a bunch of credit cards isn't too hard either), you can save a decent amount of money. Sometimes a service will be offered by a bunch of different companies, so you can rotate around through all of them.

Any digital content that you have access to for a limited amount of time can you downloaded, so you have access forever. For example, given a one-month trial on Netflix, you can download every movie or show you're interested in, enough to last for years.

File Sharing

Find some friends that have the thing you're looking for and are willing to share it. Given the ability to connect with billions of other people with the internet, it's not very hard to find some friends.

Note that the listed sites often get shutdown, so a copy with the same name but a different URL will show up to replace it.

Hosted

Books: Library Genesis
Movies/TV: Putlocker, PotLocker, Movie4k

Torrents

PirateBay.
KickassTorrents.
AudioBookBay.

Forums / Chat Rooms

For niche content, users who don't know about torrents, and/or users who like to socialize.

You can find forums with a web search, but that also means the content gets taken down quicker. Chat rooms are better because they are harder to find.

Disclaimer

Downloading stolen content is illegal, however you're unlikely to get in trouble for it. If you weren't going to purchase it anyway, then getting a free copy harms no one. If you want people to continue working on something, someone needs to pay them.
